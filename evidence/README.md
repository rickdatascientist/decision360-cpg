# Evidence ledger

Every public claim must point to one or more of:

- a public source with URL and access date;
- a sanitized interview or review summary with consent status;
- a reproducible synthetic-data experiment;
- an evaluation report with method, thresholds, failures, and limitations;
- a versioned product decision.

Never commit personal data, customer data, employer material, interview recordings, access tokens, or confidential notes. Unsupported claims must be labeled as hypotheses or removed.
