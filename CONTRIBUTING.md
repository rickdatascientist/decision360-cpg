# Contributing

This project is in evidence-gathering pre-alpha. Before proposing code, open an issue describing the buyer decision, evidence, acceptance test, failure behavior, and product boundary affected.

Contributions must use public or independently created inputs. Do not submit confidential data, employer/customer material, secrets, unverifiable performance claims, or copied code without a compatible license and provenance.

A public reuse license has not yet been selected; contribution terms will be formalized before accepting external code.
